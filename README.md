# Abhijeet Kulkarni — Portfolio

Static single-page portfolio. No Node or backend — ready for GitHub Pages.

**Repository:** [https://github.com/abhi-24395/Ahijeetwebsite](https://github.com/abhi-24395/Ahijeetwebsite)

## What’s included

- **index.html** — Single file: hero, about, skills, projects, connect form, “Ask Abhijeet” chat widget
- Contact form opens the default email app with a pre-filled message (mailto)
- Responsive layout and hamburger nav on mobile
- Design system (Outfit + Source Sans 3), dark theme, accessibility-friendly

## Host on GitHub Pages

1. Push this repo to GitHub.
2. **Settings** → **Pages** → Source: **Deploy from a branch**.
3. Branch: **main**, folder: **/ (root)**. Save.
4. Live site: **https://abhi-24395.github.io/Ahijeetwebsite/**

## Edit the site

Open **index.html** and edit the content (name, bio, projects, links, contact email). No build step.
